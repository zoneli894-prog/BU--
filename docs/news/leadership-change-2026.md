# 2026年各部门负责人换届通知

各位 BU 成员：

经主席团审议批准，现正式启动 2026 年度各部门负责人换届工作。相关事项通知如下：

## 换届范围

本次换届涉及以下部门的部长及副部长职位：

- 宣传部
- 外联部
- 活动部
- 技术部
- 财务部

## 参选条件

1. 加入 BU 满一年以上
2. 过往一年内无违纪记录
3. 具备相应的专业能力或管理经验
4. 能够保证每周投入不少于 5 小时用于部门工作

## 时间安排

| 阶段 | 时间 |
|------|------|
| 报名阶段 | 2月1日 - 2月15日 |
| 资格审核 | 2月16日 - 2月20日 |
| 竞选演讲 | 2月25日 |
| 投票选举 | 2月26日 - 2月28日 |
| 结果公示 | 3月1日 |

## 报名方式

请有意向参选的成员填写《部门负责人竞选报名表》，并于 2 月 15 日前发送至秘书处邮箱。
